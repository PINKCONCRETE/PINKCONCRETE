## Selected Work

These projects are ordered by a mix of personal contribution, completeness, public impact signals, and how strongly they represent my current direction.

| Rank | Project | Why It Leads |
| --- | --- | --- |
| 1 | [YOLO26-Tools](https://github.com/PINKCONCRETE/YOLO26-Tools) | Most complete public perception project: VLM + GroundingDINO object detection, COCO dataset generation, and YOLO fine-tuning. |
| 2 | [RoboAutoTask](https://github.com/PINKCONCRETE/RoboAutoTask) | Core robotics automation work: a Python workspace for repeatable robot task orchestration. |
| 3 | [openarm](https://github.com/PINKCONCRETE/openarm) + [openarm_eepose](https://github.com/PINKCONCRETE/openarm_eepose) | Robot control foundation around OpenArm, combining C++ integration work with Python pose experiments. |
| 4 | [SceneAnnotation](https://github.com/PINKCONCRETE/SceneAnnotation) | Data-side infrastructure for robotics perception, useful as a bridge between raw scenes and model training. |
| 5 | [robocoin-pipeline](https://github.com/PINKCONCRETE/robocoin-pipeline) | Pipeline-level robotics experiments that connect perception, automation, and execution flow. |
| 6 | [RL4Mario](https://github.com/PINKCONCRETE/RL4Mario) | A focused reinforcement-learning project that shows experimentation outside the core robotics stack. |
